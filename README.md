# ID Assignment 1 - Personal CV/Portfolio Website

## Project Overview
This repository contains a static one-page personal CV/portfolio website made with plain HTML, CSS, and a small amount of JavaScript for the slide-out navigation menu.

## Original Context
This was built as a first-year student assignment to present personal background, education, skills, and achievements in a simple scroll-based format.

> **Archived project note:** This is an archived early learning project. The goal is to preserve its original student-project character while keeping it functional and easier to read/maintain.

## Features
- Single-page portfolio layout
- Hero introduction section
- Education and skills sections
- Projects and personality highlights
- School and achievement sections
- Contact form UI with built-in HTML validation
- Social media profile links
- Mobile navigation toggle using JavaScript

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla, for menu toggle)
- No frameworks or build tools

## How to Run Locally
1. Clone or download this repository.
2. Open `index.html` directly in your browser.
3. No installation, no package manager, and no build step required.

## What Was Cleaned Up
- Fixed multiple spelling/wording issues in site text (for clarity and professionalism).
- Fixed semantic/link markup issues (for example, anchor + button nesting mistakes).
- Added safer external link behavior with `rel="noopener noreferrer"` on `target="_blank"` links.
- Improved image handling and mobile responsiveness while preserving original layout/style.
- Cleaned up CSS duplication and grouped repetitive selectors.
- Corrected responsive CSS typo that prevented one mobile layout rule from applying.
- Kept the website fully static and dependency-free.
